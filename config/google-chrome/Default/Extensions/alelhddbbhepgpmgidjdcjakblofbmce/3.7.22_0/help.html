Help & Feedback

In some cases, the extension may stop working. If you experience this issue, you can solve it with these steps:

1. Open the Chrome extension page ("chrome://extensions/" in the address bar)

2. Find "Awesome Screenshot"

3. Disable Awesome Screenshot, then re-enable it

4. Refresh the page you want to capture with Awesome Screenshot, and try to take a screenshot again

If you experience problems, or if you have other suggestions, don't hesitate to shoot us an email at screenshot+chrome@diigo.com. If you are reporting and issue, include details about how we might reproduce the issue you experienced. If you would like additional features, describe a usage scenario for a specific feature.
