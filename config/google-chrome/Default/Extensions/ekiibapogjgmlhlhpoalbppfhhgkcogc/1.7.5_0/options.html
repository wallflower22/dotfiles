The Green Web Chrome Extension

The Green Web Search:

The Green Web Search adds the different green web icons to search results for the Google, Yahoo, Bing and Ecosia search engines.
Check out the addonpage for information on the different icons and how it works.

Disabled

The Green Web for all external links:

The Green Web can also be used to check all extenal links on a page. All links to other domains that are greenly hosted will be green underlined. Hovering over those links will show a small popup with more information.
