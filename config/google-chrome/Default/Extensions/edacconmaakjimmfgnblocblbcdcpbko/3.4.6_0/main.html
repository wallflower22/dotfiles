v3.4.5Released 2015-01-16
  • You can now navigate backward and forward through the sessions you've selected within a Session Buddy tab via the browser's back/forward buttons.
  • Previous and Saved sessions are now directly accessible via (local) URL.
  • Session Buddy tab title now denotes the selected session(s).
  • Session Buddy now presents a clear set of troubleshooting instructions if it fails to start.
  • Fixed a bug causing Export to falsely omit the Current Session in some cases.
v3.4Released 2014-12-12
  • Enhanced import and export feature.
  • In-app manual backup and restore feature.
  • Incognito windows are displayed uniquely and handled appropriately across the app. Note that "Allow in incognito" must be turned on for Session Buddy in order for it to display and manage incognito windows.
  • Window state (maximized, minimized, or normal) is saved and restored accordingly.
  • Merged sessions adopt the name of the first-selected named session (if any) in the set of sessions being merged.
  • Clicking on a pin icon next to a tab title restores/selects the tab as pinned.
  • Window title tooltips show the default restore type.
  • Left/right arrow key focuses navigation/content pane.
  • Significantly improved session render performance.
  • Support for Retina and other high pixel density displays.
  • UI update. This includes a new setting to revert to Session Buddy's "classic font" if desired.
  • Rating solicitation frequency dialed down.
  • Added checks to address the generation of empty Previous Sessions.
  • Notification on "Save current session" in context menu of web pages is now working and manual refresh of open Session Buddy tabs is no longer necessary for such saved sessions to appear.
  • The first tab of a single window restore is now the correct one when the window is not the first window in its session and the default restore type is set to "one window".
  • Whitespace no longer collapses in the display of session names and tab titles.
  • Session deletion now functions properly when invoked via the session menu.
  • Search terms containing single quotes now correctly match content.
  • Session save keyboard shortcut now works when "Ask for name when saving sessions" is turned on.
  • Sessions are now merged as expected (instead of simply saved) when hitting the Enter key on the session name dialog during the course of a merge.
  • Corrected an issue preventing displayed session dates and tab counts from reverting in some cases when undoing a session edit.
  • Session edit undo invoked via keyboard shortcut no longer overwrites a newly-selected session.
  • Session names and tab titles no longer reveal HTML character entities when certain search terms are used.
  • Improved handling of special characters in session names, tab titles, and URLs.
  • Session Buddy icon click in an incognito window now delegates to a non-incognito window instead of showing a broken page.
